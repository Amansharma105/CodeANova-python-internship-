# Student Record Management System

A simple Python console application to manage student records using CRUD operations.

## Features
- Add Student Record
- View All Records
- Search by Roll Number
- Update Student Marks
- Delete Student Record
- Calculate Total & Percentage

## Tech Used
- Python 3.x
- Core Python
- Functions & Loops
- Lists & Dictionaries
- Exception Handling

## Run Project

```bash
python main.py
```
